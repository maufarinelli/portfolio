FAQ

Vous trouvez ci-dessus les questions qui sont souvent posées pour nos clients (et pour le prof aussi).

1 - Pourquoi avez-vous des liens dans le site web qui ne marchent pas?
Il existe des liens qui ne marchent pas parce que ce n'est qu'un travail pratique du cours Développement de Site Web et Commerce Électronique du Collège de Maisonneuve dont il faudrait créer un certain numéro de pages, mais ne pas concevoir le site web entier. Les pages qui ont été développées sont seulement la page d'accueil avec nos produits et les pages suivantes : Notre emplacement, Qui sommes-nous, Contactez-nous, Nous liens, Nos Partenaires et FAQ. Si on devait développer un vrai site web transactionnel, tous les liens auraient leurs pages développées et certainement les pages auraient des bases de données reliées, etc.

2 - Pourquoi il y a un message Alert de sécurité dans Adobe Flash Player?
Ce message apparaît à cause du vidéo du produit vedette de la page d'accueil. J'ai mis un lien du Youtube avec le vidéo de détails du produit afin de donner plus d'informations aux clients, mais quelques ordinateurs ont des restrictions au code qui cherche l'information ailleurs.

3- Pourquoi il y a encore des erreurs de validation dans la page index.htm?
Toutes les erreurs trouvées pour le W3 Validation Service sont des erreurs causées par le code embedded du vidéo qui vient du Youtube. J'ai décidé, quand même, de laisser le vidéo sur mon site web parce que je pense qu'il agrandit mon projet de commerce électronique.

4- Quelle est la résolution de votre site web?
En fait, j'ai décidé de faire un site web "élastique", c'est-à-dire, peut importe la résolution d'écran utilisé, la page va être toujours 98% de l'écran. Cependant, le site est optimisé pour 1024px au plus, donc à l'écran 800px il ne va pas être bien arrangé.

5- Qu'est-ce que ce le menu horizontal en haut, juste à côté du logo?
Comme j'ai dit à la question 1, si on devait développer un vrai site web transactionnel, toutes les pages seraient développées. Ce côté-là serait des liens de personnalisation, c'est-à-dire, tous les liens que seraient utilisés pour les clients qui veulent accéder à leurs comptes, leurs informations personnelles, leurs listes de désirs, leurs panier.
